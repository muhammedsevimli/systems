# How to Paste the Prospects (2 minutes)

> Command 1 gave you the links. Now you are moving the brands you see at those links into here. Nothing technical: look, copy, paste.

## Step by step
1. Click the link the system gave you. The Meta Ad Library opens (public, no account needed).
2. You see the brands **currently running ads** in that niche. Each card has the brand name at the top, the ad copy below, and the "Started running on" date in the corner.
3. Pick the brands that look right for you. For each brand, copy these:
   - the brand name
   - the ad copy (the text on top + the headline + the button text)
   - the "Started running on" date (this is the budget signal, the most important data)
   - how many different ads they are running (the library says "X results")
   - the format (single image / video / carousel)
4. Open a file in this folder (like `dental-clinic.md`, named after the niche) and paste into the template below.
5. Run the Step 2 command in `RUN.md`.

## Template (one block per brand)
```text
## <Brand name>
What they sell: <one sentence>
Started running on: <the date shown in the library>
How many ads are running: <number>
Format: <single image / video / carousel>
Ad copy:
<paste the full text of the ad here>
Button: <Learn More / Shop Now / Send Message ...>
Note: <if it is a video, one line on what it shows. otherwise leave blank.>

---
```

## Tips
- **The date is the most valuable data.** An ad running for months = it is running because it makes money = they have a budget. The system bases the ranking on this, so do not skip it.
- Do not clean up the copy, paste it as is. Even a typo is a signal.
- Leave a field blank if you are not sure. The system does not invent; if it is blank, it says it is blank.
- 4-5 brands is a good start. Few brands = weak ranking.
- If a video ad has little text, add a one-line note ("a woman takes the product out of the box"). The system does not watch the video, it uses your note.
