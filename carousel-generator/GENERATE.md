# GENERATE · One Command

> When you want a carousel, open this file and copy the command below, replacing `<topic>` with your own topic.
> Thanks to `CLAUDE.md`, Claude Code will already read the brand + voice + swipe + format files.

## Command (copy, change the topic)

```
make an instagram carousel about: <topic>

rules:
- first say in one line that you read the brand + voice + swipe + format files.
- produce 3 different hooks (pain angle, number angle, contrarian angle). each can be the cover slide.
- then write a full 7-slide carousel: cover + 5 middle slides + close (CTA).
- add a one-line "visual:" design note under each slide.
- do not use the banned words in 02-voice.md, keep the tone.
- write the output to the outputs/ folder as YYYY-MM-DD-topic.md.
- at the very end, give a LinkedIn address note (switch to the measured register).
```

## Short version (when in a hurry)
```
make a carousel about: <topic>. follow the format and voice files, 3 hooks + 7 slides, write the output to outputs/.
```

## Note
The clearer the topic, the better the output. Bad topic: "sales". Good topic: "why a small business can sell without discounting".
