# 03 · Decisions (working memory)

> This file is LIVE. It grows every week. Accumulating decisions like "let's not do that again",
> "this campaign landed", "we will feature this product" go here.
> The system follows these decisions in new output. Everything you keep in your head lands here.

## How to use
- When you make a new decision, add it at the top with the date (newest on top).
- When you ask the system for output, it reads this file and follows past decisions.
- Once a month, move stale decisions to the "archive" heading so the file does not bloat.
