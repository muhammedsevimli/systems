# Weekly Loop · 5 minutes

> This is the only thing that keeps the memory alive. Once a week, 5 minutes.
> Skip it and the system keeps remembering the old brand. Do it and the system grows with you.

## Ritual (same day every week, e.g. Monday morning)

1. **What did you learn this week?**
   A headline that landed, a message that flopped, a reaction from a customer.
   Write it at the top of `memory/03-decisions.md` with today's date.

2. **What gets featured?**
   Which product/offer/message you are pushing this week.
   Update the "What we are featuring" section in `03-decisions.md`.

3. **Any tone drift?**
   Look at the copy you produced over the past week. If a word that is not yours keeps showing up,
   add it to the BANNED list in `memory/02-voice.md`.

4. **Clear out the stale.**
   Move any decision in `03-decisions.md` that no longer holds to the "Archive" heading.

5. **Test it.**
   Have the system make a small piece ("write a single Instagram caption for this week's product").
   Is the tone right, does it know the featured product. If not, find the missing file and fill it in.

## Monthly (once a month, 15 minutes)
- Is `01-brand-identity.md` still accurate. New product, new audience, price change.
- Has `03-decisions.md` grown too big. Trim the archive.
