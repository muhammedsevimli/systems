# Input Breakdown Anatomy

> The system reads the three files in `inputs/` against these headings. If the information is not in the input, it leaves that line blank, never invents.

## 1) Winning ads (`inputs/winning-ads.md`)
Headings to extract for each winning ad:
1. **Hook:** the ad's first sentence / the scroll-stopping opener. (pain / question / number / contrarian / curiosity / humor.)
2. **Promise / offer:** what it promises. Result, ease, freshness, low risk, gift, value.
3. **For whom:** who it speaks to, from its language and pain choice. Age signal, lifestyle, moment of use.
4. **Emotion:** which emotion it pushes (relief, avoiding regret, pleasure, belonging, smart spending, trust).
5. **Format:** video / single image / carousel / UGC.
6. **Proof:** number, guarantee, review screenshot, user demonstration.
7. **How long it has been running / performance note:** if available, "best for X months", "best-selling ad". The longest-running ad is the strongest signal.

## 2) Customer reviews (`inputs/customer-reviews.md`)
The system collects these from the reviews:
- **Repeating praise:** what customers actually love (if the same thing appears in many reviews, that is your real sales angle).
- **The customer's own sentence:** the concrete phrase the customer used is the strongest ad hook. Keep it as raw material ("I can't go back to supermarket coffee" style).
- **Repeating objection:** the hesitation before buying (price, commitment, "will it work"). Every objection is a concept opportunity: a hook that answers the objection head-on.
- **The sentence that refutes the objection:** another customer's words that cancel that objection (this is the most persuasive proof).

## 3) Most-liked comments (`inputs/liked-comments.md`)
The most-liked comments on social media are the map of the viral angle:
- **Viral emotion:** humor, righteous anger, "wow" moment, acceptance. The most-liked comment is the crowd thinking out loud.
- **The unspoken truth:** the thing everyone feels but does not say (many likes = everyone agrees). Turn this into a hook.

## Pattern extraction (combine the three inputs)
After everything is broken down, the system answers these four questions:
- Which hook type keeps landing (pain / sensory / number / social proof / contrarian / humor)?
- What is the common promise type (freshness / ease / value / low risk / belonging)?
- What is the common emotion (relief, avoiding regret, pleasure, trust, smart spending)?
- What is the most common objection, and which is the best customer sentence that refutes it?
These four answers are "the logic that clearly works". The 50 concepts derive from this logic; they take the words from the customer's real voice and the logic from the winning ads.

## Honest note (tell the audience)
- The system works with the input you give: the more real and rich the input, the more accurate the output. If you put in made-up reviews or fake winning ads, the system learns the wrong pattern.
- The system does not invent customer reviews. If it uses a review in quotes, it takes it from the real reviews you provided.
- How you know a "winning ad": the best-performing (best-selling / longest-running) ads in your ad account. If you do not run ads, you can give your highest-engagement organic posts as winners.
