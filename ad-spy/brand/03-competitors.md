# 03 · Competitors (tracking list)

> The competitors whose ads you want to crack. The system builds Meta Ad Library links from this list.
> Write it once, run from the same links every month. Add new competitors as you notice them. This list is the living ad memory of your industry.

## How to fill it
For each competitor: name (as it appears in the ad/on the page), Facebook page link (if any), page number (page id, if you know it), a short note.
- **How to find the page number (page id, optional):** open the competitor's Facebook page, go to the "About / Transparency" section; the page number is written there. If you do not know it, leave it blank, the system searches by name.
- Searching by name is usually enough. The page id only matters when several brands share the same name.
