# This is where you paste competitor ads

> This folder is the system's input. In Step 1 you click the link you generated, and you paste the active ads you see in the Meta Ad Library here.

## How to paste (3 minutes)
1. Click the link the system gave you in Step 1. The Meta Ad Library opens and lists the competitor's **currently running (active)** ads.
2. Each ad card shows: the top text (the actual ad copy), the headline, the button text, and the "Published date / Running since". Select and copy this info.
3. Open a file named after the competitor in this folder (e.g. `yumag.md`) and paste the ads one under another. Separate each ad with `---`.
4. If it is a video ad: watch the video and drop a one-line note on what happens in the first 3 seconds and what is written on screen. Not required, but if you add it the concept comes out sharper.

## Paste template (for each ad)
```text
## Ad 1
Format: video / single image / carousel
Running: <e.g. 3 months / 2 weeks>
Text:
<paste the top text of the ad here>
Headline: <if any>
Button: <e.g. Shop Now / Learn More>
Video note (optional): <what happens in the first 3s, what is on screen>

---

## Ad 2
...
```

## Tips
- Paste at least 2 competitors, 2-3 ads per competitor. A pattern only comes out with more than one ad.
- Be sure to grab the long-running ads; the winning signal is there.
- Do not delete this guide file; the system only reads the competitor files, this one can stay.
