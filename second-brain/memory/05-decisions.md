# 05 · Decisions (working memory)

> This file is LIVE, it grows every week. Accumulated decisions like "let's not do this again",
> "this worked", "let's feature this now" land here.
> The system follows these decisions on new work and writes new ones back here itself.

## How to use
- When you make a new decision, add it at the top with the date (newest on top).
- When you ask the system for work, it follows past decisions.
- Once a month, move stale decisions under "Archive" so the file does not bloat.
