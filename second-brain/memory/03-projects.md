# 03 · Projects (active work memory)

> The work you are on, its status, the next step. A living file.
> When a project is done, move it under "Archive" so the active list stays clean.

## How to use
- For each project: name, for whom, status, next step, important notes.
- Update when the status changes. When you ask the system "where are we on this project" it reads from here.
- Done? Move it to the archive.

---

## Active projects

### [Project name]
- **For whom:** [customer / your own business]
- **Status:** [new / in progress / near delivery / on hold]
- **Next step:** [one clear sentence]
- **Notes:** [decision, blocker, date]

Example:
### Coffee shop website
- **For whom:** a local coffee shop
- **Status:** in progress
- **Next step:** waiting on the menu photos, will finish the home page once they arrive
- **Notes:** the customer wants a dark theme, no online ordering for now
