## Which pages should be scanned (the system tries these itself)
The system follows the order below automatically. If you have a direct link, paste it here and the system does not have to guess.
- Home page: <URL>
- Features: <URL or leave empty, the system tries /features, /product, /platform>
- Pricing: <URL or leave empty, the system tries /pricing, /plans>
- Documentation: <URL or leave empty, the system tries /docs, /help, /api>

## Scan boundary (the system obeys these rules)
- Public pages only. A page that wants a login, sits behind a paywall or requires signing up is not scanned.
- A page closed by `robots.txt` or refusing access is not scanned, it is marked "could not be reached".
- The site's design, copy and images are not copied. Only what it does is read.
